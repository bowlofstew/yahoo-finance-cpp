# Yahoo! Finance Data Downloader

This project allows for command-line downloading of Yahoo! finance data.

# Software requirements

 * Boost
 
 * CMake
 
 * A Modern C++ compiler

# Building the software

  *  mkdir build
  
  *  cd build 
  
  *  cmake ..
  
  *  make
